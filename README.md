#egjs-component

## requirement

```
// dependencies
npm install
```

## build

```
npm run build //or webpack
```

## test

```
npm run test //or karma start
```

## documentation

```
npm run jsdoc //or jsdoc -c jsdoc.json
```

## dev server

```
npm run dev-server //or webpack-dev-server
```

checkout http://localhost:8080/demo/
